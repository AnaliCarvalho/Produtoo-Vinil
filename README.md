Nesse projeto foi criado um sistema de identificação de vinis disponíveis para compra, onde irá mostrar o nome do artista, nome do álbum/vinil, ano de lançamento e o preço vendido para os clientes, e também a média de preço dos vinis. Para esse projeto utilizamos a discografia completa da cantora Taylor Swift.
Para esse projeto foi utilizado:
- Estrutura de repetição for-each
- Array list
- Getters e Setters
